=== 任務 1：探索 Linux 目錄結構 ===
執行以下指令：
ls /
bin                home               lost+found  root                srv
bin.usr-is-merged  init               media       run                 sys
boot               lib                mnt         sbin                tmp
dev                lib.usr-is-merged  opt         sbin.usr-is-merged  usr
etc                lib64              proc        snap                var

請從上面的結果中，說明以下 5 個目錄各是做什麼的：
/home：使用者的家目錄，存放每個一般使用者的個人檔案。

/etc：存放系統所有的設定檔（例如密碼檔、網路設定）。

/var：存放變動性高的檔案（例如系統日誌 Log、暫存郵件）。

/tmp：存放暫時性的檔案，系統重啟時通常會清空。

/usr：存放使用者安裝的應用程式、函式庫與執行檔（類似 Windows 的 Program Files）。

=== 任務 3：chown 與 chgrp 實作 ===
請依序執行以下指令並貼上每步的結果：

步驟 2：建立測試用的使用者和群組

sudo groupadd tempgroup
(無輸出表示成功)）

步驟 3：用 chown 改變擁有者

-rw-r--r-- 1 spes1 spes1 15 Apr 14 08:43 /home/spes1/week06_test.txt
